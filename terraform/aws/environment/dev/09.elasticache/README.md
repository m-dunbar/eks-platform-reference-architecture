# Elasticache

ElastiCache offers two caching engines — Memcached and Redis. Valkey is the open‑source, Redis‑compatible fork maintained by the Linux Foundation that AWS now supports as a lower‑cost, drop‑in alternative. It preserves Redis protocol compatibility, command behavior, and data structures while avoiding the licensing restrictions introduced in newer Redis releases.

## Further Reading

For further information regarding AWS KMS concepts, the service it provides, and it's implementation, see [AWS Elasticache Documentation](https://docs.aws.amazon.com/elasticache/).

For further information regarding Terraform's Resource configuration, see Hashicorp's [Elasticache Module Documentation](https://registry.terraform.io/modules/terraform-aws-modules/elasticache/aws/latest).
